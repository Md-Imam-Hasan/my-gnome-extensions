# `utils/` — Clipboard History formatting

Text helpers for **Clipboard History** (`clipboard@imam`).

## Files

- **`format.js`** — UTF-8 byte length checks (10 KiB cap), whitespace normalization for stored text, and single-line preview truncation for menu labels.

## Notes

- Used by [`../services/clipboardService.js`](../services/clipboardService.js) and [`../panel/dropdown.js`](../panel/dropdown.js).
